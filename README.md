# Oppgave beskrivelse

Planen er å lage en klesbutikk som tilbyr et bredt utvalgt av klær både for menn og kvinner, du skal kunne bla gjennom produkter, legge varer i handlekurven og fullføre kjøpet tryg og enkelt. Mye fokus på backend og design delen.

# Hva man skal kunne gjøre:

Man skal kunne opprette en bruker og logge seg inn

Man skal kunne legge til produkter i en handlekurv

Det skal være en checkout for betaling

En produktside der man ser info om produket, pris, etc

En produkt katalog over alle produktene


# Teknologier 

Jeg tenker å bruke html css, python, js, MariaDB, Figma og Photoshop

Html, CSS og Javascript er for frontenden 

Python og MariaDB er for backend, (MariaDB Database)
